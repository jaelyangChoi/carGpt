Chat-gpt open API 기반 차량 추천 프로그램

# 개요
자동차 구매 고객 관리 차원에서, 차량 재구매 시기가 도래하면 차량을 추천하여 광고성 이메일을 발송한다.

# 주요 기능
 1) 고객 정보 기반으로 Chat-GPT에 질의를 던져 차량을 추천 받는다.
 2) 질의 대한 응답값을 바탕으로 추천 차량에 대한 정보를 고객에게 발송한다.
